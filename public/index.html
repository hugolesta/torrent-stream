About Torrent-Stream

Torrent-Stream was conceived as an out of the box torrent client on the browser, just select the torrent that you most like and start the download of the files.
Optionally the main goal of this project is to stream video torrents on the browser, so if you have any intent to contribute with this, you're most than welcome my friend!.
This is it for now, so.. have fun!, and thanks for the fish! :)

Core Principles

Three principles:

  • Free: No need for nothing to be paid.
  • Easy: Just hit a button and start
  • Accesible: Anyone can do it and use it.
